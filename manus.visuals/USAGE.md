# DFWA Visual Usage Guide (Updated)

## Kern-Visuals (Style-Lock: 85% Black, 14% Neon Green, 1% Red)

1. **ack_splash_void.png**: Splash-Screen / Pre-Loader. Pulsierendes Auge, Binär-Ringe. Markenprägung.
2. **ack_category_realm.png**: Hintergrund für Frage-Sektoren. Mainframe-Look, Hardware/AI/Security Themen.
3. **ack_streak_evolution.png**: Dynamisches HUD/Hintergrund-Asset für Streak-Erfolge. Steigende Energie-Intensität.
4. **ack_interference_glitch.png**: Overlay bei Fehlern oder Zeitnot. SYSTEM_ERROR Glitch-Effekt.
5. **ack_reaction_set.png**: ACK Charakter-Feedback (Hohn, Skeptik, Dominanz, Fokus). UI-Pop-ups.
6. **ack_hall_of_infamy.png**: Leaderboard Platz 1. Thronender ACK mit Code-Krone. Elite-Status.
7. **ack_system_shutdown.png**: Beenden-Screen. Schließender Iris-Verschluss, finaler grüner Pixel.

## Event-Visuals
- **ack_victory.png**: Wird angezeigt, wenn der Spieler verliert (ACK siegt).
- **ack_player_win_angry.png**: Wird angezeigt, wenn der Spieler siegt (ACK ist wütend).
- **ack_pause.png**: Wird angezeigt, wenn das Spiel pausiert wird (ACK gelangweilt).
- **ack_override_alien.png**: Start einer Runde/Challenge (INIT Button).
